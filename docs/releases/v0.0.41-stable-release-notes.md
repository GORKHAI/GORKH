# GORKH v0.0.41 — Stable macOS Release

**Release Date:** 2026-04-28  
**Platform:** macOS (Apple Silicon + Intel)  
**Channel:** stable  
**Tag:** v0.0.41

---

## What Is GORKH

GORKH is a desktop-native AI agent that sees your screen, understands your workflow, and takes action on your Mac — with your explicit approval every step of the way.

---

## What's New in v0.0.41

### Stable macOS Release

This is the first stable macOS release of GORKH. The app is Developer ID signed and notarized by Apple for GateKeeper-friendly installation.

### No Local Model Installation Required

Local AI/Ollama infrastructure has been completely removed. You do not need to install Ollama, download local models, or manage a runtime.

### Hosted GORKH Free Tier

Start immediately with GORKH Free:
- 5 hosted tasks per day at no cost
- No API key required
- No account required for basic usage
- Tasks run on hosted infrastructure; your screen data stays on your Mac and is never persisted

### Bring Your Own API Keys

For unlimited usage, connect your own cloud provider API keys:
- **OpenAI** (GPT-4o)
- **Anthropic Claude** (Claude 3.5 Sonnet)
- **DeepSeek**
- **Moonshot (Kimi)**
- **MiniMax**
- **Custom OpenAI-compatible endpoint** (self-hosted vLLM, llama.cpp server, etc.)

Your API keys are stored exclusively in the macOS Keychain. They never leave your machine and are never sent to GORKH servers.

### Computer-Use Reliability

- Screenshot dimensions passed correctly to vision models
- Coordinate safety clamping (normalized [0,1] with NaN/Infinity rejection)
- Verification loop after each action (screenshot hash comparison)
- Stuck-loop detection and retry logic

## Installation

1. Download the DMG for your Mac:
   - **Apple Silicon** (M1/M2/M3/M4)
   - **Intel** (x86_64)
2. Open the DMG and drag GORKH to Applications
3. Launch GORKH from Applications
4. Grant Screen Recording and Accessibility permissions when prompted
5. Sign in or start using GORKH Free immediately

---

## System Requirements

- macOS 12+ (Monterey or later)
- Screen Recording permission
- Accessibility permission

---

## Security

- API keys stored in macOS Keychain only
- Screen captures are in-memory only (60-second expiry)
- Every sensitive action requires explicit user approval
- No server-side LLM keys
- Developer ID signed and notarized by Apple

---

## Known Limitations

- **macOS only:** Windows release is on the roadmap
- **Advanced Agent:** The experimental Rust AdvancedAgent path is not the production path. The stable release uses the production legacy AiAssistController computer-use path.
- **GORKH Free:** Limited to 5 tasks per day. Connect a BYO provider for unlimited usage.

---

## Updater

Stable releases include automatic update checking. The app checks for updates at launch and notifies you when a new stable version is available.
