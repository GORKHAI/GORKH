# GORKH v0.0.50 Stable Release Notes

## Summary

This release completes the Premium Workstation Unification pass for the desktop app and includes the recent Solana workstation modules:

- Transaction Studio v0.1
- Wallet Hub + Portfolio Dashboard
- Builder Developer Toolbox + RPC & Node Manager
- DeFi Command Center with read-only backend aggregator integration

## Premium Workstation UI

- Unified older Markets, Shield, Agent, Context, Wallet, Builder, and Private surfaces under the shared dark graphite workstation frame.
- Kept the Workstation shell fixed: sidebar, topbar, inspector, and status bar remain persistent.
- Main workspace overflow is bounded; long lists, logs, JSON, previews, inspector content, and chat histories use internal scroll panes.
- Global status copy now separates shell safety from module-local data sources.

## Safety

No new execution features were added.

Still locked or unavailable in this release:

- signing
- transaction broadcast
- swaps
- limit orders
- lending actions
- LP actions
- staking/unstaking
- program deploy/upgrade/close
- authority management
- Jito execution
- Squads execution
- hardware signing
- Drift integration
- autonomous wallet or developer execution

## DeFi Backend

The existing GORKH API now exposes read-only DeFi aggregator endpoints for health, positions, LSTs, yields, lending, LP, and Jupiter quote-only summaries. Backend env may contain public API/indexer/RPC keys, but never wallet secrets or signing material. Jupiter remains quote-only and no executable transaction payloads are returned.

## QA

The workstation QA checklist has been updated with premium UI, fixed-shell, safety-label, and manual screenshot requirements.
